# 08-hvac-terminology-pl.md - Słownik Terminologii HVAC (PL)

> **[DIRECTIVE]** Ten plik jest nadrzędnym źródłem prawdy dla nazewnictwa w interfejsie użytkownika (UI), komunikatach i dokumentacji. Zakazuje się używania kalk językowych z angielskiego.

## 1. STRUMIENIE I BILANS (AIR BALANCE)
- Supply Air -> Nawiew
- Exhaust Air -> Wywiew / Wyciąg
- Transfer Air (In/Out) -> Powietrze transferowe (Dopływ / Odpływ)
- Net Balance -> Bilans netto
- Air Change Rate (ACH) -> Krotność wymian [1/h]
- Overpressure -> Nadciśnienie
- Underpressure -> Podciśnienie
- Infiltration -> Infiltracja
- Exfiltration -> Eksyfiltracja
- Balanced -> Zbilansowany

## 2. TERMODYNAMIKA (THERMODYNAMICS)
- Heat Gain (Sensible) -> Zyski ciepła (Jawne) [W]
- Heat Gain (Latent) -> Zyski ciepła (Utajone) [W]
- Total Heat Gain -> Całkowite zyski ciepła [W]
- Room Temperature -> Temperatura w pomieszczeniu [°C]
- Supply Temperature -> Temperatura nawiewu [°C]
- Relative Humidity (RH) -> Wilgotność względna [%]
- Enthalpy -> Entalpia [kJ/kg]
- Dew Point -> Punkt rosy [°C]
- Specific Volume -> Objętość właściwa [m3/kg]

## 3. MECHANIKA PŁYNÓW (FLUID MECHANICS)
- Pressure Drop / Loss -> Strata ciśnienia / Opór [Pa]
- Friction Loss -> Strata tarcia [Pa/m]
- Dynamic Pressure -> Ciśnienie dynamiczne [Pa]
- Static Pressure -> Ciśnienie statyczne [Pa]
- Total Pressure -> Ciśnienie całkowite [Pa]
- Velocity -> Prędkość [m/s]
- Air Density -> Gęstość powietrza [kg/m3]
- Absolute Roughness -> Chropowatość bezwzględna [mm]
- Hydraulic Diameter -> Średnica hydrauliczna [mm]
- Laminar / Turbulent Flow -> Przepływ laminarny / turbulentny

## 4. ELEMENTY I SYSTEMY (UI LABELS)
- Zone / Room -> Pomieszczenie / Strefa
- Occupants -> Liczba osób
- Dose per Occupant -> Dawka na osobę [m3/h/os]
- Normative Flow -> Wydatek normatywny [m3/h]
- Calculation Mode -> Tryb obliczeń
- Target ACH -> Krotność zadana
- Real ACH -> Krotność rzeczywista
- System Supply (S) -> System Nawiewny (N)
- System Exhaust (E) -> System Wyciągowy (W)
