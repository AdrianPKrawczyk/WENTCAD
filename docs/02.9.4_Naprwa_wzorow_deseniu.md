# TASK: Naprawa Wzorów Deseniu (Hatch Patterns) na Canvasie
Główną przyczyną tego, że wzory deseniu się nie renderowały, był fakt, że generowałeś Canvasy na podstawie surowej tablicy `systems` (`sys.color`). W aplikacji wiele systemów korzysta z *automatycznej palety kolorów*, co oznacza, że ich bezpośredni atrybut `color` jest w wielu przypadkach równy `undefined`. Jeśli `sys.color` było `undefined`, funkcja `createPatternImage` w ogóle nie była uruchamiana i deseń był pusty!

Aby całkowicie usunąć ten problem i zoptymalizować wydajność, wykonaj dokładnie poniższe kroki w pliku `src/components/Workspace2D.tsx`:

### 1. Pobranie globalnych ustawień widoku ze Store'a
Na górze komponentu `Workspace2D`, tam gdzie pobierasz inne dane ze Store (`useZoneStore`), dodaj:
```tsx
const isSystemColoringEnabled = useZoneStore((s) => s.isSystemColoringEnabled);
const globalSystemOpacity = useZoneStore((s) => s.globalSystemOpacity);
2. Dynamiczny Cache dla Wzorów (Zamiast useMemo)
Usuń całkowicie blok const systemPatterns = useMemo(...). Jest on błędny, ponieważ ignoruje dynamiczne kolory mapowane przez resolveZoneStyle. Zamiast tego zdefiniuj cache oparty na hooku useRef:

TypeScript
const patternCache = useRef<Record<string, HTMLCanvasElement>>({});
3. Aplikacja wzorów i kolorów w pętli polygons.map
Wewnątrz funkcji polygons.map((poly) => { ... }) zaktualizuj logikę pobierania stylów w następujący sposób:

TypeScript
const zone = zones[poly.zoneId];
if (!zone) return null;

// Filtrowanie (pozostaw obecną logikę isSupplyHidden itp.)
// ...

// Zawsze rozwiązujemy ostateczny styl strefy za pomocą dedykowanego algorytmu
const style = resolveZoneStyle(zone, systems, globalSystemOpacity); 

// KRYTYCZNE: Używamy stylów systemowych TYLKO jeśli włączony jest Widok Systemowy
const shouldUseSystemStyle = isSystemColoringEnabled && style.color;

// Ustalenie bazowego koloru (jeśli systemowy wyłączony, dajemy domyślny błękitny)
const baseColor = shouldUseSystemStyle ? style.color : '#0ea5e9';

// Zostaw obecną logikę isChecked, isSelected, isRedefining
const isRedefining = redefiningZoneId === poly.zoneId;
const isChecked = checkedZoneIds.includes(zone.id);
const isSelected = selectedZoneId === zone.id;

let currentFill = isRedefining ? '#ef444460' : baseColor; 
// Nadanie półprzezroczystości (HEX alpha), aby zawsze było widać podkład
if (!isRedefining && !isChecked && !isSelected && shouldUseSystemStyle) {
   currentFill = baseColor + '60'; 
} else if (!isRedefining && !isChecked && !isSelected) {
   currentFill = baseColor + '40';
}

let currentStroke = isRedefining ? '#ef4444' : baseColor;
let currentStrokeWidth = (isRedefining ? 3 : 2) / scale;
let shadowProps = {};

if (!isRedefining) {
  if (isChecked) {
    currentFill = '#f59e0b80'; 
    currentStroke = '#d97706';
    currentStrokeWidth = 4 / scale;
    shadowProps = { shadowColor: '#f59e0b', shadowBlur: 15 / scale, shadowOpacity: 0.8 };
  } else if (isSelected) {
    currentFill = '#4f46e560'; 
    currentStroke = '#4338ca';
    currentStrokeWidth = 3 / scale;
  }
}

// === GENEROWANIE WZORU Z CACHE W LOCIE ===
let patternImg: HTMLCanvasElement | null = null;
if (shouldUseSystemStyle && style.patternId) {
  const cacheKey = `${style.patternId}-${baseColor}`;
  if (!patternCache.current[cacheKey]) {
    const newPattern = createPatternImage(style.patternId, baseColor);
    if (newPattern) patternCache.current[cacheKey] = newPattern;
  }
  patternImg = patternCache.current[cacheKey] || null;
}
4. Renderowanie warstw w Konva
Podwójna warstwa <Line> wewnątrz <Group> jest prawidłowym podejściem, ale wymaga użycia nowej zmiennej patternImg wyciągniętej w KROKU 3. Upewnij się, że struktura w return dla pojedynczego poligonu wygląda tak:

TypeScript
return (
  <Group key={poly.id}>
    {/* Warstwa Tła (Zawsze widoczna) */}
    <Line
      points={poly.points}
      closed={true}
      fill={currentFill}
      stroke={currentStroke}
      strokeWidth={currentStrokeWidth}
      dash={isRedefining ? [5 / scale, 5 / scale] : []}
      opacity={isRedefining ? 0.8 : 1}
      {...shadowProps}
      onMouseEnter={...} // Zostaw obecne eventy
      onMouseLeave={...}
      onClick={...}
    />
    
    {/* Warstwa Deseniu (Tylko w Widoku Systemowym i gdy przypisano wzór) */}
    {!isRedefining && patternImg && (
      <Line
        points={poly.points}
        closed={true}
        fillPatternImage={patternImg as any}
        fillPatternRepeat="repeat"
        fillPatternScale={{ x: 1 / scale, y: 1 / scale }}
        listening={false} // Przebija kliknięcia w dół, by można było klikać w strefy
      />
    )}
  </Group>
);
